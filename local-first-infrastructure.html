The Wrong vs Right Question

The wrong question: Why move AI to local?
The real question: What does dependency actually cost?

The API Death Spiral

Everything routed through APIs.

It worked. Until it scaled.

Cost #1: Scaling Breaks First

100 req/day → ₹150/day 1,000 req/day → ₹1,500/day 100,000 req/day → ₹1,50,000/day

You don’t control growth. The API bill controls you.

  • Shut it down
  • Pay ₹40L+/month
  • Break UX
No fourth option.

Cost #2: Lock-in

  • No control over models
  • No control over pricing
  • No control over behavior
You don’t own the system.

Cost #3: Margin Collapse

Revenue - API cost = margin

Better product → more calls → less margin.

APIs punish success.

What Changed

  • Mac Mini: ₹65,000

Owning became cheaper than renting.
